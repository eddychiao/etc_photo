HERO / LANDING PAGE BACKGROUND
================================

Place your hero image here with the filename:
  hero.jpg  (or .jpeg / .webp / .png)

Then update the background-image reference in views/index.ejs if needed.

Recommended specs:
  - Minimum resolution: 1920 x 1080px
  - Format: JPEG (compressed) or WebP for best performance
  - Orientation: landscape or portrait both work
